# Examples for ATAS ‚Äì Academic Text Analysis System

This folder contains **real data and outputs** generated by the ATAS software (Academic Text Analysis System).  
The dataset used comes from the **official transcript of the Brazilian Chamber of Deputies** during the **2016 impeachment session of President Dilma Rousseff**.

The files here demonstrate the **full workflow** of ATAS: text preprocessing, lexical filtering, statistical analysis, sentiment evaluation, and network generation (Gephi).

## üìä 2. Output Tables (Generated by ATAS)

| File | Description |
|------|--------------|
| `tabela.csv` | Bigrams extracted from the full text. |
| `tabela_locais.csv` | Named entities recognized as **locations**. |
| `tabela_nomes.csv` | Named entities recognized as **people**. |
| `tabela_organizacao.csv` | Named entities recognized as **organizations**. |

| `estatistica_sentimento.csv` | Preliminary sentiment analysis results for each speaker. |
| `Nos.csv` | Node table exported from Gephi (5178 nodes). |
| `Arestas.csv` | Edge table exported from Gephi (21961 edges). |
